# Contributing

Thank you for considering contributing to this project.

## How to Contribute

There are many ways to contribute, including:

- Reporting bugs
- Suggesting new features
- Improving documentation
- Submitting code changes
- Reviewing pull requests

## Before You Start

Please:

1. Check existing issues and pull requests before creating a new one.
2. Open an issue first for major changes so the discussion can happen before work begins.
3. Make sure your contribution aligns with the goals of the project.

## Development Setup

1. Fork the repository
2. Clone your fork
3. Create a new branch for your change

Example:

```bash
git checkout -b feature/your-feature-name
```
